# Module 9A - Build Your First React Component

## Description
This is a React app created using Vile that renders a UserProfile component. The component includes a name, bio, and image styled with inline CSS.

## What I learned
- How to set up a React project using Vite
- How to write and structure a functional React component
- How to use JSX and inline styles
- How to Render a component in App.jsx

## Challenges
To be honest, I didnt encounter any challenges with this.
